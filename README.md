# Hidrosum · Lavandería Autoservicio 🫧

Sitio web oficial de **Hidrosum**, lavandería autoservicio en Pachuca, Hidalgo, México.
*Tu lavandería + amigable.*

## ✨ Características

- Sitio estático 100% autocontenido (HTML + CSS + JS vanilla), sin build ni dependencias.
- Identidad de marca completa: degradado morado → azul → turquesa, burbujas animadas, ondas de agua y tipografía redondeada (Baloo 2 + Nunito).
- Animaciones: burbujas flotantes generadas por JS, reveal on scroll, ondas SVG, marquee, hover states e íconos animados.
- Secciones: hero, servicios, cómo funciona, programa Hidropuntos, galería con lightbox, horarios y ubicación con mapa de Google, FAQ y CTA.
- Responsive (móvil, tablet, escritorio), menú hamburguesa y soporte de `prefers-reduced-motion`.
- SEO: metadatos Open Graph y datos estructurados `LocalBusiness` (schema.org).

## 🚀 Desarrollo local

No requiere instalación. Abre `index.html` en el navegador o sirve la carpeta:

```bash
npx serve .
```
